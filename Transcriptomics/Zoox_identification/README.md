## Zooxantellae identification
Reads, that did not align to *O. patagonica* genome, were extracted to fasta on the previous step with `--outReadsUnmapped Fastx` argument in [mapping.basic.sh](https://github.com/talimass/Ocullina_Translocation/blob/main/Transcriptomics/QC_Mapping/mapping.basic.sh) script. After that, salmon v.0.14.1 was used to map unaligned reads to the proteome database of available symbionts species created [earlier](https://github.com/talimass/Cayman-translocation/tree/main/RNAseq_analysis/Zoox_identification) during _P. astreoides_ translocation project. Counts of reads of each symbiont species was calculated for each sample by R script [symbio.salmon.Op.R](https://github.com/talimass/Ocullina_Translocation/blob/main/Transcriptomics/Zoox_identification/symbio.salmon.Op.R).
